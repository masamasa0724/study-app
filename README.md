⚛️ 原子力発電基礎 勉強アプリ

## 概要

このWebアプリは、原子力発電の基礎知識を学ぶための一問一答形式の学習ツールです。全25問の問題を通じて、原子力発電の仕組みから安全性、課題まで、幅広い知識を習得できます。

## 特徴

✨ **25問の充実した問題集**
- 原子力発電の基本原理
- 原子炉の構造と動作
- 安全対策と課題
- 放射能と被曝
- 原子力と環境

📊 **学習進捗の管理**
- 進捗率の表示
- 回答履歴の自動保存
- 正解数の追跡

📚 **詳細な解説付き**
- 各問題に対する丁寧な解説
- 理解を深めるための背景知識

🎯 **インタラクティブな評価**
- クイズ完了時に成績評価
- フィードバックメッセージ
- 結果に応じた学習アドバイス

## 使い方

1. **ブラウザで開く**
   - `index.html` をWebブラウザで開いてください

2. **スタートボタンをクリック**
   - 「スタート」ボタンで学習を開始します

3. **問題に回答**
   - 4つの選択肢から正解を選びます
   - 回答後に解説が表示されます

4. **進めていく**
   - 「次へ」ボタンで次の問題に進みます
   - 「前へ」ボタンで前の問題に戻ります

5. **結果確認**
   - 全問題完了後に成績が表示されます
   - フィードバックに基づいて学習を続けられます

## 機能詳細

### 学習画面
- 現在の問題番号と総問題数を表示
- 正解数の表示
- 進捗バーのリアルタイム更新
- 選択肢の明確なハイライト表示

### 解説機能
- 回答後に詳細な解説を表示
- 正解・不正解の視覚的フィードバック
- 何度でも解説を確認可能

### 進捗保存
- ローカルストレージに自動保存
- ページを閉じても進捗が保持される
- リセットボタンで進捗をクリア可能

### 結果評価
- 成績に基づいた5段階のフィードバック
- 学習を続けるためのアドバイス
- 再挑戦オプション

## 問題の内容

### 基本原理（1-5問）
- エネルギー源
- 核分裂反応
- 冷却と蒸気化
- タービン発電
- 制御棒

### 利点と課題（6-10問）
- 利点と課題
- 放射能と放射線
- 安全対策
- 熱エネルギー変換

### 安全と環境（11-15問）
- 冷却停止時の危険
- 蒸気排出と放射能
- 使用済み燃料
- 半減期
- 発電方式の比較

### 現代的課題（16-25問）
- 建設費と期間
- 被曝の種類
- 核燃料サイクル
- プルトニウム
- CO2排出量
- 福島事故
- 廃棄物処分
- 都市部供給
- IAEA
- 高速炉
- 発電効率
